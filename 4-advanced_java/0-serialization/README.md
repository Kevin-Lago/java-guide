# Serialization

## Serializable

> Serialization is a mechanism of converting the state of an object into a byte stream. Deserialization is the reverse process where the byte stream is used to recreate the actual Java object in memory. This mechanism is used to persist the object. - [Geeks for Geeks](https://www.geeksforgeeks.org/serialization-in-java/)
